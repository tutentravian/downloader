خیلی ساده‌تر از چیزی‌ست که فکر می‌کنی. برای استفاده از پروژه professional-downloader، تنها کاری که باید بکنی این است:

۱. پروژه را فورک (Fork) کن

· اول برو به صفحه اصلی پروژه (professional-downloader).
· دکمه Fork (بالای صفحه، سمت راست) رو بزن.
· صفحه‌ای باز میشه - فقط کافیه دکمه سبز رنگ Create fork رو بزنی.
· همین! الان یک کپی کامل از پروژه توی اکانت خودت داری.

۲. دسترسی نوشتن را فعال کن (فقط یک بار!)

این یک مرحله بسیار کوتاه است و اگر انجام ندهی، فرآیند دانلود کار نخواهد کرد. چون فایل دانلود شده باید در ریپازیتوری تو ذخیره شود (عملیات Write)، باید اجازه را صادر کنیم.

· در صفحه ریپازیتوری فورک شده‌ی خودت، به تب Settings برو.
· از منوی سمت چپ، وارد قسمت Actions و سپس General بشو.
· به پایین اسکرول کن تا به بخش Workflow permissions برسی.
· گزینه Read and write permissions را انتخاب کن.
· دکمه Save رو بزن. (کار تمام شد!)

*برای اطلاعات بیشتر، می‌توانی به مستندات رسمی گیت‌هاب در این زمینه مراجعه کنی.*[reference:3]

۳. ازش استفاده کن!

حالا هر موقع خواستی فایلی دانلود کنی:

· به تب Actions در مخزن خودت برو.
· روی نام workflow (که معمولا Smart File Downloader است) در سمت چپ کلیک کن.
· دکمه آبی رنگ Run workflow رو بزن.
· لینک فایل، رمز (اختیاری) و حجم تکه‌ها رو وارد کن.
· دکمه سبز رنگ Run workflow رو بزن. تمام!

خلاصه که: برو صفحه پروژه را فورک کن، بعد در Settings فقط یک بار دسترسی Write رو فعال کن. همین! بقیه کارها با خود workflow انجام می‌شه. امیدوارم این آموزش برات مفید بوده باشه.
